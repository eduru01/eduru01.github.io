NYC Business Growth and Property Value Analysis

  • Type: Data Analysis • Regression Modeling • Urban Economics
  • Technologies: R, Regression Models (Logistic Regression, Random Forest, Decision Tree), Forward Stepwise Selection, Feature Selection
  • Course: Foundation of Data Analytics
Analyzing how the onset of new businesses, along with multiple demographic and urban factors, affects property values by zip code in Brooklyn, New York from 2020-2025. The analysis is incorporating factors including population, emergence of unique businesses by borough, crime rate, public school locations, transit access, grocery stores, income, and other urban characteristics. Utilizing multiple regression models including logistic regression, random forest, and decision trees with forward stepwise selection to identify key factors influencing property values. The project involves data cleaning, exploratory data analysis, feature selection using PCA, and model comparison to determine the best predictive approach.
