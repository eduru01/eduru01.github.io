NYC Property Value Analysis

  • Type: Data Analysis • Regression Modeling • Urban Economics
  • Technologies: R, RStudio, tidyverse (dplyr, ggplot2), sf (spatial data), broom, readr
  • Course: Foundation of Data Analytics
Analyzed how neighborhood characteristics influence residential property values across New York City using a ZIP-code–level dataset and multiple linear regression. Integrated spatial amenities (subway stations, grocery stores, schools, crime, new businesses) with socioeconomic factors (income, education, unemployment, population) using R and the sf package. Built full and borough-specific models to compare predictors across Manhattan, Brooklyn, Queens, the Bronx, and Staten Island. Identified income and grocery access as consistent positive predictors and revealed how the drivers of housing value vary sharply across boroughs.

What I Did

  • Built a ZIP-code–level dataset by merging 10+ public datasets using R
  • Performed spatial joins to assign subway stations, crime incidents, grocery stores, schools, and businesses to ZIP codes
  • Engineered socioeconomic features (income, education, unemployment, population)
  • Built full and borough-specific multiple linear regression models
  • Evaluated model performance and interpreted coefficients to understand property value drivers
  • Synthesized findings into a final presentation and written report
