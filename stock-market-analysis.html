Analyzing how macroeconomic indicators influence U.S. stock indices using machine learning and data analytics

Stock Market Economic Indicators Analysis

  • Type: Machine Learning • Data Analysis • Financial Modeling
  • Technologies: Python, Pandas, Scikit-learn, Matplotlib, Seaborn, Kaggle Datasets
  • Course: Python and Machine Learning
This project analyzes how key macroeconomic indicators—interest rate, inflation rate, housing index, and unemployment rate—influence the behavior of major U.S. stock indices: S&P 500, NASDAQ, and Dow Jones. Using historical economic data sourced from Kaggle, I performed exploratory data analysis to uncover relationships and visualize long-term market trends. The project applies multiple machine learning models including K Nearest Neighbors (KNN), K-Means Clustering, Random Forest, Decision Tree, Logistic Regression, and Principal Component Analysis (PCA) to forecast stock index movements based on macroeconomic factors.
