Agios Pharmaceuticals Internship

  • Role: Data Sciences Intern
  • Technologies: C#, SQL Server Database, Visual Studio 2019, Microsoft SQL Server Management Studio, xUnit Testing
During my Data Sciences Internship at Agios Pharmaceuticals, I worked on three key projects that addressed critical system reliability and data management challenges. The internship focused on process improvement, system integration, and implementing automated solutions to enhance operational efficiency. Through these projects, I developed expertise in C# development, SQL Server database management, and automated testing methodologies while learning to break down complex problems into actionable solutions.

Technical Overview

The internship utilized Microsoft technologies for data management, application development, and quality assurance. The primary technologies used were:

  • C# - Application logic and automation
  • SQL Server Database - Data management and validation
  • Visual Studio 2019 - Development environment
  • Microsoft SQL Server Management Studio - Database administration
  • xUnit Testing - Automated unit testing framework

Projects

During the internship, I worked on three main projects:

  • Egnyte UM (User Management) - Automated offboarding of inactive users from the Egnyte Research System
  • CMS Barcodes (Compound Management System) - Validation and migration of compound barcodes to the new system
  • Unit Testing (Automated Testing Framework) - Implementation of automated unit tests using xUnit to ensure code reliability

Detailed information about each project, including timelines, tools, and implementation details, can be found in the presentation below.

Challenges & Learnings

Before starting, I had no experience with SQL or C#, so I self-taught both languages in preparation for the role which was a steep learning curve that taught me how to learn complex technologies quickly and independently.

During the internship, I faced the challenge of managing multiple concurrent projects while adapting to shifting priorities in a fast-paced environment. This required me to develop strong time management skills, create clear project timelines, and learn when to pivot without losing momentum.

Through these challenges, I gained hands-on experience in professional software development, system integration, and implementing automated solutions. More importantly, I learned how to break down ambiguous problems into actionable steps and communicate technical solutions to non-technical stakeholders.
